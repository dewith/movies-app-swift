# Movies

A SwiftUI iOS app for browsing trending and top-rated movies and TV shows, built on top of [TMDB](https://www.themoviedb.org/) data.

## Features

- Home tab with hero title and horizontally scrolling lists for trending/top-rated movies and TV shows
- Tab-based navigation: Home, Upcoming, Search, Download

## Requirements

- Xcode 16+
- iOS 18+

## Getting Started

1. Clone the repo
2. Open `Movies.xcodeproj` in Xcode
3. Build and run on a simulator or device
